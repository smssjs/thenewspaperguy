Browser Storage

Keep your subscriptions

Feeds are auto-saved in this browser. If your browser clears site data, private mode is enabled, or you switch devices, export a backup and import it later.

Backups are plain JSON files containing your configured feed names, URLs, categories, columns, and hidden-state settings.
Google Drive Sync

Sync across devices

Sign in with Google and store your subscription configuration in your Google Drive app data folder. The same Google account can then load the same feed setup on any device.

You must create a Google OAuth web client and authorize this GitHub Pages origin before Drive sync will work. Your feed file is stored privately in Drive appDataFolder, not in visible folders.
Firebase MVP

Auth, subscriber tracking, and cloud profile

This layer adds Google sign-in through Firebase Auth, Firestore-backed cloud subscriptions, basic subscriber metrics, and a donation button. Paste your Firebase web config JSON from the Firebase console to activate it.

Discover by Interest
